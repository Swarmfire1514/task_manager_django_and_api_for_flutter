Jan 16, 2026

Progress:
- Explored Django templates vs DRF:

Django Templates:
- Server renders HTML; tightly coupled backend and frontend
- Requires cookies, sessions, CSRF protection
- Good for simple websites, admin panels, content pages
- Drawback: Not reusable for mobile apps

Django API (DRF):
- Backend returns JSON; frontend is separate
- JWT authentication for mobile/web clients
- Good for web apps, mobile apps, multi-client setups
- Advantage: reusable, scalable, modern

Project Implementation:
- Website uses Django template rendering
- Flutter app consumes APIs from DRF
- Hybrid approach shows strengths and limitations of both
